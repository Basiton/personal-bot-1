# MAX Messenger Bot

A simple and extensible bot for MAX Messenger with referral system, user management, and admin panel.

## Features

- ✅ User registration and management
- ✅ Referral system with points
- ✅ SQLite/PostgreSQL database support
- ✅ FastAPI admin panel
- ✅ Async/await pattern
- ✅ Easy configuration via .env
